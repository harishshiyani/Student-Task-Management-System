# Student-Portfolio-Management-System
Student Portfolio Management System using PHP, HTML, CSS, JavaScript and MySQL

## Technologies
- PHP
- HTML5
- CSS3
- JavaScript
- MySQL

## Features
- Student Registration
- Login & Logout
- Dashboard
- Portfolio Management
- Profile Update
- Change Password

## Installation
1. Clone the repository.
2. Import the SQL file into MySQL.
3. Copy the project to the htdocs folder.
4. Start Apache and MySQL in XAMPP.
5. Open http://localhost/Student-Portfolio-Management-System
